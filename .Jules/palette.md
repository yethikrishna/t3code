## 2024-05-20 - Add aria-label to thread rename input\n**Learning:** Inline inputs without visible `<label>` tags (common for in-place edit features like renaming a thread in the sidebar) are inaccessible to screen readers without an explicit `aria-label` attribute.\n**Action:** Whenever introducing inline editing functionality, ensure an `aria-label` is provided if a visible `<label>` is not used.
